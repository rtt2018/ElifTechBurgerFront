# Delivery Food App



## Level of accomplished tasks - base + some functions from middle and advance level 

backend: https://github.com/rtt2018/ElifTechBurger

Виконані функції: 
- Бекенд виконаний з використанням MongoDB
- Функція додавання страв у корзину: можна збільшувати/зменшувати кількість, видаляти позицію
- Вибір магазину
- Сортування товарів за ціною (asc, desc) та назвою (a-z, z-a), як всіх, так і з вибраного магазину
- пагінація: кнопка Load More
- Функція Reorder - можливість перезамовити будь яке попереднє замовлення 
- Сторінка перегляду історії замовлень
